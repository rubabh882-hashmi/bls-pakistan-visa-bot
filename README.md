# BLS Pakistan Visa Bot

This repository contains a bot for automating the visa application process for Pakistan. It aims to simplify the user experience when applying for visas through the BLS International services.

## Features
- Automated filling of visa application forms.
- Tracking of application status.
- Notifications on application progress.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rubabh882-hashmi/bls-pakistan-visa-bot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd bls-pakistan-visa-bot
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
- Run the bot by executing:
  ```bash
  python bot.py
  ```
